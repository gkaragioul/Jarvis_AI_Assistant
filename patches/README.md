# Patch Snapshots

This directory stores versioned overlay snapshots for the Mac OpenJarvis app.
They are intended to document or replay Jarvis-specific changes against a
separate upstream OpenJarvis checkout.

The upstream OpenJarvis project is not vendored in this repository. Patch files
may still contain OpenJarvis source context and derived modifications, so the
upstream Apache-2.0 license and notice obligations remain relevant for that
material.

Do not treat this directory as MIT-only. The repository-level MIT License
covers original Jarvis_AI_Assistant material, but it does not relicense
OpenJarvis-derived patch content or any third-party dependency.
